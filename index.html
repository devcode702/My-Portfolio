01

Customer Churn Analysis

End-to-end churn analysis on a 50,000-user music streaming dataset

Excel SQL Power BI
Inactivity = #1 churn signal
50,000 users analyzed
Interactive Power BI dashboard
View on GitHub

Problem

Music streaming platforms struggle to identify which users are about to leave — and why — before it's too late.

Data Work

  • Cleaned & preprocessed 50,000-user dataset in Excel
  • Validated and queried data patterns using SQL
  • Segmented users by activity, engagement & skip behavior

Key Insights

  • Inactivity is the strongest & earliest predictor of churn
  • Engagement metrics had minimal impact on churn rate
  • Skip behavior & listening time nearly identical across users

Outcome

  • Built an interactive Power BI dashboard for churn monitoring
  • Churn is not sudden — it builds over time through inactivity
  • Delivered actionable retention strategy recommendations
